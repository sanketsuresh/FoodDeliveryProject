# Online Food Delivery System 🍽️

A simple web-based food ordering system using **Java**, **JSP**, **MySQL**, and **Apache Tomcat**.

## Features:
- User Registration and Login
- Food Menu Display
- Add to Cart and Place Order
- Admin Dashboard to View All Orders

## Technologies Used:
- Java, JSP, HTML, CSS
- MySQL for Database
- Apache Tomcat for Deployment

## How to Run:
1. Import the project in Eclipse or keep it in your Tomcat `webapps` folder.
2. Configure database in `db.jsp` and start the server.
3. Access at `http://localhost:8080/DeliveryFood/`.
