# 🍽️ Online Food Delivery System

This is a simple Java + JSP + MySQL-based web application for ordering food online. Users can register, login, view menu, add items to cart, and place orders. Admin can view all orders placed by users.

---

## 👨‍💻 Tech Stack

- Java
- JSP (JavaServer Pages)
- MySQL
- Apache Tomcat
- HTML/CSS

---

## 📁 Features

- ✅ User Registration & Login
- ✅ View Food Menu
- ✅ Add to Cart & Place Orders
- ✅ Admin Dashboard to View All Orders

---

## 🛠️ How to Run

1. Install Apache Tomcat and MySQL.
2. Create a database `fooddb` and import tables:
   - `users` (email, password, name)
   - `orders` (user_email, item_name, price)
3. Place project in `webapps` folder of Tomcat.
4. Start the Tomcat server.
5. Open `http://localhost:8080/DeliveryFood/`.
